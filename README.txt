Instalacja
----------
1. Przejdź do katalogu projektu:
    cd xkcd-comics-viewer

2. Zainstaluj zależności:
    npm install

Uruchomienie
------------
1. Uruchom aplikację:
    npm start

Wymagania
---------
- Node.js
- npm
- Przeglądarka internetowa
- Plik `nr.txt` zawierający numery komiksów
